# Build tools for Liberland documents

## Requirements
- Pandoc
- PDFLatex

How to build:
```
$ git clone https://github.com/burningtree/liberland-doc-tools.git
$ cd liberland-doc-tools
$ make
```

All generated documents are placed in the `dist/` directory.
